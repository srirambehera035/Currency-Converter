# Currency Converter

## Project Overview

Currency Converter is a Java-based console application that allows users to convert currencies using either real-time exchange rates fetched from an external API or offline fallback exchange rates stored within the application. The project is designed to provide accurate, fast, and user-friendly currency conversion while demonstrating concepts such as API integration, data structures, exception handling, and menu-driven programming in Java.

The application supports multiple international currencies and offers several useful features including single currency conversion, batch conversion, exchange rate viewing, conversion history tracking, and the ability to switch between online and offline modes. This ensures that users can continue using the application even when internet connectivity is unavailable.

## ✨ Key Features
- 🔄 Real-Time Currency Conversion
  - Fetches live exchange rates using the ExchangeRate API.
  - Provides accurate and up-to-date conversion results.
  - Supports dynamic exchange rate retrieval based on the selected base currency.
- 📴 Offline Conversion Support
  - Includes pre-defined fallback exchange rates for major global currencies.
  - Allows uninterrupted currency conversion when the API is unavailable.
  - Ensures reliability even without an internet connection.
- 💱 Single Currency Conversion
  - Convert any supported currency into another.
  - Displays:
    - Converted amount
    - Exchange rate
    - Inverse exchange rate
    - Data source used (Live or Offline)
- 📊 Exchange Rate Viewer
  - View exchange rates of multiple currencies relative to a selected base currency.
  - Helps users compare currency values efficiently.
- 🚀 Batch Currency Conversion
  - Convert a single amount into multiple target currencies simultaneously.
  - Useful for travelers, traders, and financial analysts.
- 📝 Conversion History
  - Stores the last 10 currency conversions.
  - Enables users to review recent conversion activities quickly.
- 🌍 Multiple Currency Support

Supports popular currencies including:

    - USD (US Dollar)
    - EUR (Euro)
    - GBP (British Pound)
    - INR (Indian Rupee)
    - JPY (Japanese Yen)
    - CAD (Canadian Dollar)
    - AUD (Australian Dollar)
    - SGD (Singapore Dollar)
    - AED (UAE Dirham)
    - And many more.
- 🎨 User-Friendly Console Interface
  - Interactive menu-driven navigation.
  - Clean formatting with currency symbols.
  - Input validation and error handling for a better user experience.

## 🎯 Project Objective
The main objective of this project is to provide a practical and efficient currency conversion system while helping developers understand how real-time APIs can be integrated into Java applications. It combines networking, data processing, and user interaction into a single application, making it an excellent project for learning intermediate Java programming concepts.
